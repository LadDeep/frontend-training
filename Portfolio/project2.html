Just A Minute!

Just A Minute(JAM) is a game based on the BBC's game show. This game is aimed at improving communication skills of users by providing rapid-fire questions to which they have to speak for a minute. The game even provide feedback loop improve by listening their speeches.

There are basically two modes:

  1. Classic
  2. Custom

Technologies Used

  • Java
  • Android
  • Firebase Auth
  • Firebase Firestore
