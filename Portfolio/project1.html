GraphicsAI

GraphicsAI is being developed with the aim of providing users with a platform to create and manipulate 3d models on-the-fly.

Platform enables users to create as well as modify models, textures and bluring images. Harnessing the power of AI would porvide users with better suggestions based on their activities.

There are three major parts of this tool:

  1. 3d Graphics Editor
  2. Texture Creator
  3. Image Manipulator

Technologies Used

  • HTML
  • CSS
  • JS
  • Three.js
  • React.js
  • Python
  • OpenCV
  • Pixellib
